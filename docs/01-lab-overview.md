\# Lab Overview



This repository contains a scripted Windows Server lab environment simulating a small enterprise network. The focus is on automation, repeatability, documentation, and practical systems administration skills.



\## Objectives



\- Build a functional Active Directory domain rapidly and consistently

\- Provide a professional OU structure and baseline GPOs

\- Serve as a foundation for additional labs such as:

&nbsp; - VPN and secure remote access

&nbsp; - Centralized logging and monitoring (SIEM)

&nbsp; - Patch and configuration management

&nbsp; - Hybrid identity with Entra ID and Intune



\## Scope



Included:

\- Domain controller

\- DNS and DHCP configuration

\- OU and GPO structure

\- Deployment and validation documentation



Not included (but possible extensions):

\- Firewall/security appliance configuration

\- VPN deployment

\- Endpoint imaging and application packaging

\- Cloud services (Azure, M365)
